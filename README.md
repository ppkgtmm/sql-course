## Set up

1. Install [PostgreSQL](https://www.postgresql.org/download/)
2. Go to `db-setup` folder

```sh
cd db-setup
```

3. Grant permission to `setup.sh`

```sh
chmod +x setup.sh
```

4. Run `setup.sh` to prepare database

```sh
./setup.sh
```

5. Exit `db-setup` folder

```sh
cd ../
```
